directives                        | html attribut со специальным префиксом v- , указывающий на выполнение dom монипуляций с элементом
  • V-TEXT - для манипуляции текстовой нодой
    {{ greatings }}
  • V-HTML - для вставки внутрь тега html разметку
  • V-BIND - для вставки аттрибута
    v-bind:width - для вставки аттрибута width и его значения. Сокращенно :width="50+'px'"
    :src - для вставки аттрибута :src="'./img/' + vueIcon" vueIcon мы подтянули с data
  • V-BIND:CLASS
    :class - с помощью boolean мы даем или убираем этот класс
    Давайте дадим этому элементу класс red - name_class, classRed - boolean value in data
  • V-BIND:STYLE - задать стили через аттрибут style
    подтягиваем стили с json
    Stylized text var2 - 'color: ' + 'red' Stylized text var1 - { color: 'red' } Stylized text from cssDynamic.string Stylized text from cssDynamic.object
  • V-SHOW, V-IF, V-MODEL
    Переключение экранов с помощью
    Display block v-show="display" if data.display:true Content template v-if="!display" if data.display:false

  • V-ON
    one click - trigger classRed(true,false), its add and remove class red
